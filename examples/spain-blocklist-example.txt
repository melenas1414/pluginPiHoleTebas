# === NOTAS IMPORTANTES ===
#
# 1. USO LEGAL SOLAMENTE:
#    Este plugin debe usarse ÚNICAMENTE para:
#    - Restaurar acceso a servicios bloqueados colateralmente
#    - Investigación académica sobre censura
#    - Análisis de red corporativo
#    ❌ NO para acceder a contenido pirata
#    ❌ NO para evadir bloqueos legales
#
# 2. FUENTES PÚBLICAS:
#    Los usuarios deben obtener listas de fuentes públicas legítimas:
#    - Organizaciones de derechos digitales
#    - Wikis sobre censura en internet
#    - Listas comunitarias verificadas
#    - Estudios académicos publicados
#
# 3. RESPONSABILIDAD:
#    Los usuarios son responsables del uso de este plugin
#    y deben cumplir con las leyes locales e internacionales.
#
# 4. CONFIGURACIÓN:
#    Para usar tu propia lista, configura la URL en:
#    /etc/pihole/plugins/warp/warp-config.conf
#    
#    SPAIN_BLOCKLIST_URLS=https://tu-lista-publica.com/blocklist.txt
#
# 5. VERIFICACIÓN:
#    - Verifica la fuente de tus listas
#    - Usa solo fuentes confiables
#    - Revisa periódicamente las actualizaciones
#    - Monitorea los logs del plugin
#
# === RECURSOS ===
#
# Para más información:
# - Lee la documentación: docs/SPAIN_BLOCKLISTS.md
# - Revisa el README principal del proyecto
# - Consulta organizaciones de derechos digitales
#
# === EJEMPLO DE USO ===
#
# Si mantienes tu propia lista pública en GitHub:
#
# 1. Crea un repositorio con tu lista
# 2. Publica la lista como archivo .txt
# 3. Obtén la URL raw de GitHub
# 4. Configura la URL en warp-config.conf
# 5. El plugin descargará y procesará automáticamente
#
# Ejemplo de URL:
# https://raw.githubusercontent.com/usuario/repo/main/spain-blocklist.txt
#
